# Тест-план для сайта "Papa Pizza"
## Проект GitHub: (http://papapizza59.ru/)

---

## 1. ЦЕЛИ ТЕСТИРОВАНИЯ

### Основные цели:
-  Проверка корректности работы основного функционала сайта Papa Pizza
-  Выявление дефектов работающего сайта
-  Обеспечение качества пользовательского опыта
-  Проверка кроссбраузерной совместимости

---

## 2. ОБЪЕМ ТЕСТИРОВАНИЯ

### Входит в тестирование:

| № | Область тестирования | Элементы |
|---|---------------------|----------|
| 1 | **Шапка сайта** | Логотип, навигационное меню, кнопки разделов |
| 2 | **Навигационная панель** | Пицца, Напитки, Горячие закуски, Комбо, Соусы |
| 3 | **Карточка товара** | Выбор размера, количество, добавление в корзину |
| 4 | **Фильтры** | Фильтр по цене |
| 5 | **Подвал сайта** | Пиццерия, Доставка, Оплата, Политика конфиденциальности |
| 6 | **Комбо-предложения** | Чекбоксы выбора пицц |
| 7 | **UI элементы** | Кнопка "Наверх", переключатель сетки, быстрый просмотр |
| 8 | **Формы** | Email-форма подписки |
| 9 | **Социальные сети** | Кнопка VK |

### Не входит в тестирование:
- Оформление заказа
- Мобильная версия сайта
- Нагрузочное тестирование
- Оплата

---

## 3. СТРАТЕГИЯ ТЕСТИРОВАНИЯ

| Вид тестирования | Описание | Инструменты |
|-----------------|----------|-------------|
| **Функциональное** | Проверка работы всех функций | Selenium + pytest |
| **Кроссбраузерное** | Проверка в разных браузерах | Chrome, Edge |
| **Юзабилити** | Оценка удобства интерфейса | Ручное тестирование |
| **Регрессионное** | Автоматизированная проверка | pytest + Allure |
